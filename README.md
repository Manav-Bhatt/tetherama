# Tetherama

Local-first audio-to-art visualization studio (Next.js + Tailwind + Canvas)

## Quick start

Install dependencies:

```bash
npm install
```

Run in development:

```bash
npm run dev
```

Type-check without emitting:

```bash
npx tsc --noEmit
```

Build for production:

```bash
npm run build
```

## Features

- Native Web Audio FFT analyzer
- Canvas-based generative visuals (swarm / nebula / monolith)
- Neon Wave OS-inspired UI shell and control dock

## Notes

- Use an up-to-date Chromium-based browser for best audio autoplay behavior.
- If you run into TypeScript configuration issues, check `tsconfig.json`.
